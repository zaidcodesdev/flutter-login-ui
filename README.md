# Flutter UI Card Design

A simple Flutter UI project showcasing a modern profile card design.

## ✨ Features

- Modern Card UI
- Network Image
- Custom Icons
- Gradient Follow Button
- Responsive Layout
- Clean Flutter Code

## 🛠 Tech Stack

- Flutter
- Dart
- Material Design


## 🚀 Getting Started

```bash
git clone https://github.com/zaidcodesdev/flutter-ui.git
```

Run the project:

```bash
flutter pub get
flutter run
```
